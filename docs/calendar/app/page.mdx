# @h6s/calendar

## Logic-first Calendar Hook for React

**Your calendar, your way.**

@h6s/calendar is a headless calendar library that provides pure logic without any UI. Build any calendar interface you want with complete styling freedom.

## Quick Start

```bash
npm install @h6s/calendar
# or
yarn add @h6s/calendar
# or
pnpm add @h6s/calendar
```

```tsx
import { useCalendar } from '@h6s/calendar';

function MyCalendar() {
  const { headers, body, navigation, view } = useCalendar();

  return (
    <div>
      {/* Your custom UI here */}
    </div>
  );
}
```

## Why Headless?

Headless components separate logic from presentation, giving you:

- **Complete Design Freedom** - Style it exactly how you want
- **Framework Agnostic** - Works with any CSS solution
- **Smaller Bundle Size** - No unnecessary UI code
- **Better Maintainability** - Logic and UI evolve independently
