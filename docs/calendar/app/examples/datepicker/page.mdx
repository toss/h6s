# DatePicker Examples

DatePicker implementations using @h6s/calendar with different CSS frameworks.

## Available Implementations

Choose a CSS framework to see the full implementation:

### [Tailwind CSS](/examples/datepicker/tailwind)

Modern utility-first CSS framework with built-in dark mode support.

**Features:**
- Utility-first approach
- Built-in dark mode with `dark:` variants
- Highly customizable
- Small bundle size with PurgeCSS

[View Tailwind Example →](/examples/datepicker/tailwind)

---

### [Vanilla CSS](/examples/datepicker/vanilla)

Plain CSS with no framework dependencies.

**Features:**
- No extra dependencies (besides @h6s/calendar and date-fns)
- Dark mode with `@media (prefers-color-scheme: dark)`
- Full control over styles

[View Vanilla CSS Example →](/examples/datepicker/vanilla)

---
