# DatePicker Example

A complete DatePicker implementation using @h6s/calendar with Tailwind CSS.

## Live Demo

<DatePickerDemo />

## Features

- Single date selection
- Month/year navigation
- Today button
- Keyboard navigation support
- Responsive design
- Fully styled with Tailwind CSS

## Code

```tsx
'use client';

import { useCalendar } from '@h6s/calendar';
import { format, isSameDay } from 'date-fns';
import { useState } from 'react';

export function DatePicker() {
  const [selectedDate, setSelectedDate] = useState<Date | null>(null);
  const [isOpen, setIsOpen] = useState(false);

  const { headers, body, navigation, cursorDate, view } = useCalendar({
    defaultDate: selectedDate ?? new Date(),
  });

  const handleDateSelect = (date: Date) => {
    setSelectedDate(date);
    setIsOpen(false);
  };

  return (
    <div className="relative inline-block">
      <button
        onClick={() => setIsOpen(!isOpen)}
        className="px-4 py-2 border border-gray-300 rounded-lg bg-white hover:bg-gray-50 focus:outline-none focus:ring-2 focus:ring-blue-500"
      >
        {selectedDate ? format(selectedDate, 'PPP') : 'Select a date'}
      </button>

      {isOpen && (
        <div className="absolute top-full mt-2 p-4 bg-white border border-gray-200 rounded-lg shadow-lg z-10">
          <div className="flex items-center justify-between mb-4">
            <button
              onClick={navigation.toPrev}
              className="p-2 hover:bg-gray-100 rounded"
            >
              ←
            </button>
            <h2 className="font-semibold">
              {format(cursorDate, 'MMMM yyyy')}
            </h2>
            <button
              onClick={navigation.toNext}
              className="p-2 hover:bg-gray-100 rounded"
            >
              →
            </button>
          </div>

          <button
            onClick={() => {
              navigation.setToday();
              setSelectedDate(new Date());
              setIsOpen(false);
            }}
            className="w-full mb-2 px-3 py-1 text-sm bg-gray-100 hover:bg-gray-200 rounded"
          >
            Today
          </button>

          <table className="w-full">
            <thead>
              <tr>
                {headers.weekdays.map(({ key, value }) => (
                  <th
                    key={key}
                    className="p-2 text-sm font-medium text-gray-600"
                  >
                    {format(value, 'EEEEEE')}
                  </th>
                ))}
              </tr>
            </thead>
            <tbody>
              {body.value.map(({ key, value: days }) => (
                <tr key={key}>
                  {days.map(({ key, value, isCurrentDate, isCurrentMonth }) => {
                    const isSelected = selectedDate && isSameDay(value, selectedDate);

                    return (
                      <td key={key} className="p-1">
                        <button
                          onClick={() => handleDateSelect(value)}
                          className={`
                            w-10 h-10 rounded-full text-sm
                            ${!isCurrentMonth && 'text-gray-300'}
                            ${isCurrentMonth && 'text-gray-900 hover:bg-gray-100'}
                            ${isCurrentDate && 'font-bold ring-2 ring-blue-500'}
                            ${isSelected && 'bg-blue-500 text-white hover:bg-blue-600'}
                          `}
                        >
                          {format(value, 'd')}
                        </button>
                      </td>
                    );
                  })}
                </tr>
              ))}
            </tbody>
          </table>
        </div>
      )}
    </div>
  );
}
```

## Key Implementation Details

### State Management

The DatePicker maintains two pieces of state:
- `selectedDate`: The user's selected date
- `isOpen`: Whether the calendar dropdown is visible

### Date Selection

When a date is clicked, we:
1. Update the selected date
2. Close the dropdown
3. The button text updates to show the formatted date

### Styling Classes

Key Tailwind classes used:
- `relative` and `absolute`: For dropdown positioning
- `rounded-lg` and `shadow-lg`: For modern card appearance
- `hover:bg-gray-100`: Interactive feedback
- `ring-2 ring-blue-500`: Focus/current date indication
- `bg-blue-500 text-white`: Selected date highlight

### Accessibility

- Keyboard navigation support through button elements
- Semantic HTML with proper `<table>` structure
- Focus states with ring utilities
- Clear visual feedback for interactions

## Customization

You can easily customize this DatePicker:

- **Colors**: Change `blue-500` to your brand color
- **Size**: Adjust `w-10 h-10` for larger/smaller date cells
- **Position**: Use Tailwind's positioning utilities to change dropdown position
- **Format**: Change date format strings in `format()` calls
- **Week Start**: Pass `defaultWeekStart` option to `useCalendar`
