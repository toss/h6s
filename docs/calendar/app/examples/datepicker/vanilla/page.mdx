## Key Implementation Details

### Dark Mode Support

Dark mode is handled using CSS media queries:

```css
@media (prefers-color-scheme: dark) {
  /* Dark mode styles */
}
```

The component automatically adapts based on the user's system preference.

### No Framework Dependencies

This implementation requires:
- ✅ Only `@h6s/calendar` and `date-fns`
- ✅ Plain CSS (no Tailwind, no CSS-in-JS)
- ✅ Works in any React project

## Customization

You can customize this picker by updating the CSS:

- **Colors**: Change hex color values in the CSS
- **Spacing**: Adjust padding and margin values
- **Sizing**: Modify width/height values
- **Transitions**: Change transition duration and timing
