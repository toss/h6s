## Features

- Single date selection
- Month/year navigation
- Today shortcut
- Uses Bootstrap 5 utility classes
- Dark mode support (automatic system theme detection)
- Accessible with proper ARIA labels

## Key Implementation Details

### Loading Bootstrap Styles

The live demo renders inside a lightweight `<iframe>` that loads the Bootstrap CDN stylesheet, so the docs theme stays untouched while Bootstrap's CSS variables remain available. In your app, import Bootstrap once globally (for example `import 'bootstrap/dist/css/bootstrap.min.css';` in `layout.tsx`) and render `<DatePicker />` directly.

### Dark Mode Support

This component uses Bootstrap 5's semantic color utilities (`text-body-emphasis`, `text-body-secondary`, etc.) which automatically adapt to dark mode. To enable dark mode in your app:

**Option 1: Auto-detect system theme (recommended)**

Add this script to your app's HTML `<head>` or root layout:

```html
<script>
  function updateTheme() {
    const isDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
    document.documentElement.setAttribute('data-bs-theme', isDark ? 'dark' : 'light');
  };
  updateTheme();
  window.matchMedia('(prefers-color-scheme: dark)').addEventListener('change', updateTheme);
</script>
```

**Option 2: Manual theme toggle**

```html
<html data-bs-theme="light">
  <!-- Toggle programmatically -->
  <script>
    document.documentElement.setAttribute('data-bs-theme', 'dark');
  </script>
</html>
```

### Button States

- Selected date uses `btn-primary`
- Today's date gets an outline highlight with `btn-outline-primary`
- Current month days use `text-body` for proper contrast
- Other month days fade with `text-body-secondary` and reduced opacity

## Customization

You can adapt the component by tweaking Bootstrap classes:

- Swap `btn-primary` with your theme color
- Adjust spacing utilities (`mb-3`, `py-1`, etc.)
- Configure typography with `fw-semibold`, `h6`, or custom classes
- Add icons from Bootstrap Icons for navigation arrows
