## Key Implementation Details

### Loading Bootstrap Styles

The live demo wraps `DatePicker` with a shadow-root helper that injects the Bootstrap CDN stylesheet so the docs theme stays intact. In your app, import Bootstrap once globally (for example `import 'bootstrap/dist/css/bootstrap.min.css';` in `layout.tsx`) and render `<DatePicker />` directly.

### Button States

- Selected date uses `btn-primary`
- Today's date gets an outline highlight
- Disabled month days fade using `text-muted`

## Customization

You can adapt the component by tweaking Bootstrap classes:

- Swap `btn-primary` with your theme color
- Adjust spacing utilities (`mb-3`, `py-1`, etc.)
- Configure typography with `fw-semibold`, `h6`, or custom classes
- Add icons from Bootstrap Icons for navigation arrows
