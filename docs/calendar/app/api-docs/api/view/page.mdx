# view

The `view` object provides controls for switching between different calendar view types and managing view-related settings.

## Type

```tsx
view: {
  // Current state
  type: CalendarViewType;
  isMonthView: boolean;
  isWeekView: boolean;
  isDayView: boolean;

  // Setters
  setViewType: (type: CalendarViewType) => void;
  setWeekStartsOn: (day: WeekDayType) => void;
  showMonthView: () => void;
  showWeekView: () => void;
  showDayView: () => void;
}

enum CalendarViewType {
  Month = 'month',
  Week = 'week',
  Day = 'day',
}

type WeekDayType = 0 | 1 | 2 | 3 | 4 | 5 | 6;
```

## View Types

### Month View
- Displays full month grid (4-6 weeks)
- Shows dates from previous/next months to complete weeks
- Default view type

### Week View
- Displays single week (7 days)
- Useful for weekly planners

### Day View
- Displays single day
- Useful for detailed day schedules

## Properties

### `type`

Current view type as a string.

```tsx
const { view } = useCalendar();

console.log(view.type); // 'month' | 'week' | 'day'
```

### Boolean Helpers

Convenience boolean flags for checking current view:

```tsx
const { view } = useCalendar();

view.isMonthView  // true if in month view
view.isWeekView   // true if in week view
view.isDayView    // true if in day view
```

## Methods

### `setViewType(type)`

Set view type programmatically using the enum:

```tsx
import { CalendarViewType } from '@h6s/calendar';

const { view } = useCalendar();

view.setViewType(CalendarViewType.Month);
view.setViewType(CalendarViewType.Week);
view.setViewType(CalendarViewType.Day);
```

### `showMonthView()` / `showWeekView()` / `showDayView()`

Convenience methods to switch views:

```tsx
const { view } = useCalendar();

view.showMonthView();  // Switch to month view
view.showWeekView();   // Switch to week view
view.showDayView();    // Switch to day view
```

### `setWeekStartsOn(day)`

Change which day the week starts on:

```tsx
const { view } = useCalendar();

view.setWeekStartsOn(0);  // Sunday (default)
view.setWeekStartsOn(1);  // Monday
view.setWeekStartsOn(6);  // Saturday
```

**WeekDayType values:**
- `0` = Sunday
- `1` = Monday
- `2` = Tuesday
- `3` = Wednesday
- `4` = Thursday
- `5` = Friday
- `6` = Saturday

## Usage Examples

### View Type Switcher

```tsx
import { useCalendar } from '@h6s/calendar';

function Calendar() {
  const { view } = useCalendar();

  return (
    <div className="view-switcher">
      <button
        onClick={view.showMonthView}
        disabled={view.isMonthView}
        className={view.isMonthView ? 'active' : ''}
      >
        Month
      </button>
      <button
        onClick={view.showWeekView}
        disabled={view.isWeekView}
        className={view.isWeekView ? 'active' : ''}
      >
        Week
      </button>
      <button
        onClick={view.showDayView}
        disabled={view.isDayView}
        className={view.isDayView ? 'active' : ''}
      >
        Day
      </button>
    </div>
  );
}
```

### Week Start Day Selector

```tsx
function CalendarWithWeekStartSelector() {
  const { view } = useCalendar();

  return (
    <select
      value={view.weekStartsOn}
      onChange={(e) => view.setWeekStartsOn(Number(e.target.value))}
    >
      <option value={0}>Sunday</option>
      <option value={1}>Monday</option>
      <option value={6}>Saturday</option>
    </select>
  );
}
```

### Conditional Rendering Based on View

```tsx
function AdaptiveCalendar() {
  const { view, body } = useCalendar();

  return (
    <div>
      {view.isMonthView && (
        <div className="month-grid">
          {/* Full month calendar grid */}
        </div>
      )}

      {view.isWeekView && (
        <div className="week-schedule">
          {/* Week schedule with hourly slots */}
        </div>
      )}

      {view.isDayView && (
        <div className="day-agenda">
          {/* Detailed day agenda */}
        </div>
      )}
    </div>
  );
}
```

## View Type Impact on Body Data

The `body` structure changes based on the view type:

```tsx
const { body, view } = useCalendar();

// Month view: 4-6 weeks
if (view.isMonthView) {
  console.log(body.value.length); // 4-6 weeks
  console.log(body.value[0].value.length); // 7 days per week
}

// Week view: 1 week
if (view.isWeekView) {
  console.log(body.value.length); // 1 week
  console.log(body.value[0].value.length); // 7 days
}

// Day view: 1 day
if (view.isDayView) {
  console.log(body.value.length); // 1 week (for structure consistency)
  console.log(body.value[0].value.length); // 1 day
}
```

## Default View Type

Set the initial view type when initializing:

```tsx
import { CalendarViewType } from '@h6s/calendar';

const { view } = useCalendar({
  defaultViewType: CalendarViewType.Week
});

// Calendar starts in week view
console.log(view.isWeekView); // true
```

## Navigation Behavior

Navigation methods (`toNext`, `toPrev`) adapt to the current view type:

```tsx
const { navigation, view } = useCalendar();

// Month view
view.showMonthView();
navigation.toNext(); // → Moves to next month

// Week view
view.showWeekView();
navigation.toNext(); // → Moves to next week

// Day view
view.showDayView();
navigation.toNext(); // → Moves to next day
```
