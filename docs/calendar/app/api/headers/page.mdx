# headers

The `headers` property provides structured data for rendering calendar weekday headers.

## Type

```tsx
headers: {
  weekdays: Array<{
    key: string;        // Unique key for React rendering
    value: Date;        // Date object for the weekday
  }>;
}
```

## Usage

The `headers` object contains a `weekdays` array with 7 elements (one for each day of the week). Each element includes a unique `key` for React list rendering and a `value` Date object representing that weekday.

```tsx
import { useCalendar } from '@h6s/calendar';
import { format } from 'date-fns';

function Calendar() {
  const { headers } = useCalendar();

  return (
    <thead>
      <tr>
        {headers.weekdays.map(({ key, value }) => (
          <th key={key}>
            {format(value, 'EEE')} {/* Sun, Mon, Tue, ... */}
          </th>
        ))}
      </tr>
    </thead>
  );
}
```

## Formatting Options

You can format the weekday headers using different `date-fns` format patterns:

```tsx
// Full weekday names
{format(value, 'EEEE')} // Sunday, Monday, Tuesday, ...

// Abbreviated (3 letters)
{format(value, 'EEE')} // Sun, Mon, Tue, ...

// Short (2 letters)
{format(value, 'EEEEEE')} // Su, Mo, Tu, ...

// Single letter
{format(value, 'EEEEE')} // S, M, T, ...
```

## Week Start Configuration

The weekday order respects the `defaultWeekStart` option:

```tsx
// Week starts on Sunday (default)
const { headers } = useCalendar({
  defaultWeekStart: 0
});
// headers.weekdays[0] = Sunday

// Week starts on Monday
const { headers } = useCalendar({
  defaultWeekStart: 1
});
// headers.weekdays[0] = Monday
```

## Example with Styling

### Tailwind CSS

```tsx
<thead>
  <tr>
    {headers.weekdays.map(({ key, value }) => (
      <th
        key={key}
        className="p-2 text-sm font-medium text-gray-500 dark:text-gray-400"
      >
        {format(value, 'EEEEEE')}
      </th>
    ))}
  </tr>
</thead>
```

### Vanilla CSS

```tsx
<thead>
  <tr>
    {headers.weekdays.map(({ key, value }) => (
      <th key={key} className="calendar-weekday">
        {format(value, 'EEEEEE')}
      </th>
    ))}
  </tr>
</thead>
```

```css
.calendar-weekday {
  padding: 0.5rem;
  font-size: 0.875rem;
  font-weight: 500;
  color: #6b7280;
  text-align: center;
}
```
