# Tailwind CSS

Input field that opens a date range picker popover with dual calendars, powered by `@radix-ui/react-popover`.

## Features

- Field-style trigger with placeholder and clear action
- Floating popover built with `@radix-ui/react-popover`
- Dual calendar layout showing consecutive months
- Date range selection with visual highlighting
- Hover preview showing potential range
- Synchronized month navigation
- Calendar interaction via `@h6s/calendar`
- Keyboard support for `Escape` to dismiss
- Tailwind CSS styling with dark mode variants
