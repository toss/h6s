# Tailwind CSS

## Basic

Input field that opens a date range picker popover with dual calendars, powered by `@radix-ui/react-popover`.

<Sandpack
  files={filesPicker}
  dependencies={{
    "@radix-ui/react-popover": "^1.1.2",
  }}
  providerOptions={{
    externalResources: ["https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"],

/>

## Features

- Date range picker with field trigger and popover
- Calendar-only components for inline use (basic and dual layouts)
- Field-style trigger with placeholder and clear action
- Floating popover built with `@radix-ui/react-popover`
- Dual calendar layout showing consecutive months
- Date range selection with visual highlighting
- Tailwind CSS utility classes
- Dark mode support with `dark:` variants
