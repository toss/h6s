# Bootstrap 5

## Date Range Picker

Input field that opens a date range picker popover with dual calendars, powered by `@radix-ui/react-popover`.

## Features

- Date range picker with field trigger and popover
- Calendar-only components for inline use (basic and dual layouts)
- Uses Bootstrap 5 utility classes and components
- Date range selection with visual highlighting
- Responsive design with Bootstrap grid system
- Accessible with proper ARIA labels
