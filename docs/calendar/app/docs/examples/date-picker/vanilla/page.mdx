## Features

- Date picker with field trigger and popover
- Calendar-only component for inline use
- Radix Popover gives accessible overlay behavior (focus trap, escape to close)
- Calendar generated by `@h6s/calendar` with month navigation and today highlight
- Component-scoped CSS classes
- Dark mode support with `@media (prefers-color-scheme: dark)`
